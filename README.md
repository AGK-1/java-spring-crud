# java-spring-crud
# Java Spring CRUD Application

![Java Logo](https://cdn.jsdelivr.net/gh/devicons/devicon/icons/java/java-original.svg)

## Описание
CRUD (Create, Read, Update, Delete) приложение на Java Spring Framework.

## Технологии
- **Java 11+**
- **Spring Boot 2.7+**
- **Spring Data JPA**
- **MySQL/PostgreSQL**
- **Maven/Gradle**
- **Spring Security** (опционально)

## Быстрый старт
in master
```bash
# Клонировать репозиторий
git clone https://github.com/AGK-1/java-spring-crud.git

# Собрать проект
mvn clean install

# Запустить приложение
mvn spring-boot:run
